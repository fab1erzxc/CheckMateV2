# Виджет баланса на главной странице

## What to build

Добавить на Home страницу виджет текущего баланса долга. Сейчас Home — это просто 3 кнопки (Text Entry, Photo Entry, View Receipts) без информации о долге.

Виджет загружает `GET /api/balance`, показывает число и направление долга (кто кому должен), и содержит кнопку/ссылку на страницу Balance для деталей и settlement'ов.

## Acceptance criteria

- [ ] Home страница загружает `GET /api/balance` при монтировании
- [ ] Виджет отображается сверху или между кнопками
- [ ] Показывает сумму долга и направление (user_owes_girlfriend / girlfriend_owes_user / settled)
- [ ] Есть кнопка "Перейти к балансу" или ссылка на `/balance`
- [ ] Если баланс = 0 / settled, показывается зелёный статус
- [ ] Если баланс > 0, показывается сумма и направление долга
- [ ] Обработка ошибок загрузки (скрыть виджет или показать "--")

## Blocked by

None — can start immediately
